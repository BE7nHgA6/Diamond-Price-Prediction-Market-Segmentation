# Diamond-Price-Prediction-Market-Segmentation
Diamond Dynamics is a Machine Learning project developed to predict diamond prices and classify diamonds into different market segments based on their physical and quality attributes.

The project uses data preprocessing, feature engineering, regression models, Artificial Neural Networks (ANN), and K-Means clustering to analyze diamond characteristics such as carat, cut, color, clarity, and dimensions.

An interactive Streamlit web application is also developed to:

Predict diamond price in INR
Identify the market segment of a diamond
Provide real-time predictions based on user inputs
